# 🔮 十神 ShiShen

### 你脑子里不止一个你。

你纠结的时候，有个声音说"做就是了"，另一个说"再想想"。
你分手的时候，有个声音说"断干净"，另一个说"你吃了吗"。
你一直以为这是你在跟自己吵架——

**不是。是你的十神在开会。**

十神是八字命理里的 10 个人格原型。你的八字决定了每个声音的音量——根据十神权重，**最完整地映射你自己**。

输入生辰八字 → 排出你的十神格局 → 十个"内心声音"上线议事厅 🗣️

七杀旺的人，决断力压过一切，犹豫对他们来说是 bug。
正印旺的人，共情先于逻辑，骂完人自己先心疼。
食神旺的人，快乐就是第一生产力，不开心的班不上。
伤官旺的人，真话脱口而出，得罪人是副作用。

这个插件把它们**显化**出来——装进你的 Claude，让它们在每次对话里七嘴八舌。你终于能看见自己脑子里那个会议室了。

灵感来自《极乐迪斯科》的内心声音系统 🎮 东方版，用八字驱动。

> ⚠️ **纯娱乐项目。** 写 bug 赖七杀，加薪谢正财。

## 📦 安装

```bash
claude plugin add shishen
```

或手动：

```bash
git clone https://github.com/peggy-daddy/shishen.git
cd shishen && claude plugin link .
```

## ❓ FAQ

**算得准吗？** 🤔
不准？`/shishen:fix` 直接改。我们的目标是有趣，不是准确。

**这是认真算命吗？** 🙃
不是。分手赖七杀，吵架谢伤官。一切解释权归命运所有。（并不。）

**会不会每次都出现很烦？** 😅
`/shishen:off`，他们就去喝茶了 ☕ 想他们了再 `/shishen:on`。
