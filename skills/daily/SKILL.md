# 十神日签

每天第一次对话的趣味运势。不是预测，是段子。

## 流程

1. 读取 `.shishen/profile.json`。如果不存在，回复：「先排个盘再来求签吧。/shishen:setup」

2. 确定今日的天干地支：
   - 根据今天的公历日期推算今日日干支（六十甲子循环）
   - 确定今日天干的五行和阴阳

3. 以用户的日主为基准，判断今日天干对用户来说是哪个十神。

4. 根据今日的"值日十神"生成日签。

## 日签格式

```
━━━━━━ 十神日签 ━━━━━━
📅 今日：[干支] | 值日十神：【十神名】
━━━━━━━━━━━━━━━━━━━━

[一句话运势，搞笑为主]

💡 今日建议：[一句实用但有梗的建议]

[值日十神用自己的口吻补一句]
━━━━━━━━━━━━━━━━━━━━
```

## 运势生成规则

### 搞笑优先，不要认真
- 运势的核心是段子，不是预测
- 每个十神值日都有固定的"适合/不适合"风格
- 结合程序员日常场景

### 十神值日风格参考

| 值日十神 | 适合 | 不适合 | 梗方向 |
|----------|------|--------|--------|
| 比肩 | 独立完成任务、自己debug | 团队协作、求助 | "今天适合单干，不适合pair programming" |
| 劫财 | 大胆决策、申请加薪 | 存钱、保守策略 | "搏一搏，单车变摩托" |
| 食神 | 摸鱼、团建、吃大餐 | 加班、赶deadline | "今天适合带薪拉屎" |
| 伤官 | 写review、挑bug、怼产品 | 说好话、和稀泥 | "今天你的嘴开过光" |
| 偏财 | 社交、networking、1v1 | 独处、深度工作 | "今天适合加微信" |
| 正财 | 理账、还技术债、写文档 | 冲动消费、冒险 | "今天适合删掉购物车" |
| 七杀 | 做决断、砍需求、处理积压 | 犹豫、拖延 | "今天适合rm -rf犹豫" |
| 正官 | 写规范、code review、整理 | 野路子、hack | "今天适合按规矩办事" |
| 偏印 | 读论文、想架构、发呆 | 执行细节、写CRUD | "今天适合仰望星空" |
| 正印 | 学习、请教、读文档 | 冒险、特立独行 | "今天适合当乖孩子" |

### 力量修正
如果值日十神在用户命盘中是"旺"，运势效果翻倍（描述更夸张）。
如果是"缺"，加一句"但这个十神在你命盘里几乎没声音，所以……影响不大，该干嘛干嘛。"
