# 查看命盘

读取 `.shishen/profile.json` 文件。

如果文件不存在，回复：「你还没有排过盘。输入 /shishen:setup 开始排盘。」

如果文件存在，展示：

## 格式

```
┌──────────────────────────────────────┐
│  年柱：{year.stem}{year.branch}      │
│  月柱：{month.stem}{month.branch}    │
│  日柱：{day.stem}{day.branch}        │
│  时柱：{hour.stem}{hour.branch}      │
│  日主：{dayMaster}（{dayMasterElement}）│
└──────────────────────────────────────┘
```

然后展示十神分布图（按力量从高到低排列）：

- 旺(3+)：用 `████████████` 表示
- 中(2)：用 `████████` 表示
- 弱(1)：用 `████` 表示
- 缺(0)：用 `░░░░` 表示

最后用 2-3 句轻松的话描述用户的人格特点。结合十神格局给出有趣的概括。

例如：「你是个正印旺的人——内心住着一位操心的老母亲，凡事都要替你想周到。偏财也不弱，说明你骨子里还是个社交达人。七杀几乎缺席……所以你可能不太擅长当机立断。」
